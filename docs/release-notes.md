# Release-Notes

 - Configure TnT with a .yml or .properties file.

 - Annotation @Trace to log a method.
 
 - Inoformations that will be logged by default:

	* Method name
	* Class Name
	* Execution Time
	* TimeStamps
	* Log Level
	* log message
	* correlation id
	* geoip.
	
 - HttpRest informations that will be logged:

	* Method name
	* Class Name
	* Execution Time
	* TimeStamps
	* Log Level
	* log message
	* correlation id
	* geoip
	* Http Verb
	* Http Status
	* Http URI


 - Static Fields
 
 - Nombre of threads is configurable
